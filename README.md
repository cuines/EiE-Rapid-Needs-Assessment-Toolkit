# Education in Emergencies (EiE) Rapid Needs Assessment Toolkit

## Purpose
This toolkit provides field teams with a structured approach to conduct rapid needs assessments for education programming in crisis-affected contexts. It is designed to be lightweight, adaptable, and easy to deploy in emergency settings where time and resources are limited.

## Intended Audience
- Field officers and education coordinators working in humanitarian response
- Local NGOs and community-based organizations
- Education cluster/sector coordination teams
- Government education officials in crisis-affected regions
- Researchers and evaluators focusing on education in emergencies

## Open-Source Nature
This toolkit is released as open-source under the [MIT License](LICENSE). We encourage contributions, adaptations, and translations to ensure the resource remains relevant across diverse contexts. The collaborative development model allows practitioners worldwide to share experiences and improve the toolkit based on real‑world use.

## How to Use This Repository
1. **Assessment‑Guide.md** – Contains the core methodology, stakeholder mapping, data points, and ethical considerations.
2. **Field‑Checklist.md** – A practical checklist for data collectors to ensure key information is captured during field visits.
3. **Issues** – Used to track enhancements, bug reports, and discussion about new features.
4. **Pull Requests** – Welcome contributions that add new sections, improve existing content, or adapt the toolkit to specific contexts.

## Getting Started
Clone the repository and review the `Assessment‑Guide.md` to understand the assessment framework. Use the `Field‑Checklist.md` as a companion during field missions.
